# Security Policy

This project follows a coordinated vulnerability disclosure process.

Do not report security issues in public GitHub issues.

Report vulnerabilities here:

- https://github.com/Avanishk05/secure-torch/security/advisories/new

Maintained policy details are in `docs/security_policy.md`.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Response Targets

- Initial acknowledgment: within 48 hours
- Critical fix target: within 7 days
